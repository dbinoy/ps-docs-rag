# FX Drafts Documentation Summary

  • Overview: FX Drafts is a Payment Solutions application enabling authorized in-branch creation and printing of negotiable instruments drawn on Convera partner bank accounts, supporting 20+ foreign currencies including USD.
  • Stakeholder Separation: Central 1 owns enrollment, Client Centre user management, and settlement; Convera owns application support, draft stock fulfillment, transaction processing, and technical infrastructure—escalation path is critical for support routing.
  • System Requirements: Browser-based application with validated support for Chrome 88.0+, Firefox 78.7+, and Microsoft Edge 89.0+; browser compatibility issues escalate to Convera per their user guides.
  • Key Contact Points: Central 1 Client Support (1-888-889-7878 or ServiceNow) for enrollment/settlement; Convera Client Relationship Managers (1-800-897-6333 Opt1) for transaction cancellation; Draft Fulfillment Team for stock ordering; Technical Support (Opt2) for system issues with documented hours (Mon-Thu 5am-4pm PT, Fri 5am-3pm PT).
  • Exception Handling: Voids, refunds, and stop payments route through Convera's dedicated team (draftvoid@convera.com) with structured email format requirements (same-day vs. standard void subject lines, Order ID, Draft #, currency/amount, destruction confirmation).
  • Compliance Dependencies: PCMLTFA compliance requirements differ by province (BC vs. ON); FINTRAC (Financial Transactions and Reports Analysis Centre) owns identification, record keeping, and reporting rules; both referenced via Operations Manual Program (subscription-based).
  • Documentation Artifacts: User guides segmented by role (Form 9182 for CU Admin Users, Form 9183 for Order Creator Users); remote print testing guidelines (Form 9515) available; customization guidance provided (Form 6299)—all Convera-provided documents distributed through Payment Solutions Manual.
