# Campaign Manager Summary

  • Overview: Campaign Manager is a marketing campaign management application for online banking environments, with platform-specific availability constraints.
  • Platform Dependency: Exclusively available on MemberDirect® Integrated or Forge Online Banking platforms; not supported on other banking platform implementations.
  • Functional Scope: Manages online banking marketing campaigns—specific capabilities and campaign types are detailed in the referenced Campaign Manager User Guide (not provided in this excerpt).
  • Documentation Gap: This page provides minimal technical detail; implementation architects must reference the dedicated Campaign Manager User Guide for data structures, APIs, configuration parameters, and integration workflows.
  • Integration Point: Operates as a subsystem within the MemberDirect® Integrated and Forge platforms; architects must understand how campaign data flows between Campaign Manager and the host banking platform.
  • Known Constraints: Limited availability across platform offerings suggests licensing, feature-flag, or architectural constraints that should be clarified during solution design.
  • Pre-Design Action: Review the Campaign Manager User Guide and platform-specific integration documentation before designing campaign functionality enhancements or third-party integrations.
