# Payment Solutions Manual – March 2022 Release Summary

- **Purpose**: Documentation of three content revisions made to the Payment Solutions Manual in March 2022, affecting mobile deposit agreements, AFT permissions terminology, and FTP user request contact information.

- **Mobile Deposits Constraint**: Section 3.5 (Agreements for Mobile Deposits) had content removed from Release 22:03 due to non-applicability to mobile deposit processing workflows—architects should verify current mobile deposit agreement requirements are not documented elsewhere.

- **AFT Permissions Alignment**: PaymentStream AFT User Request form (ID 6257) terminology in the AFT Permissions section was revised to match User Management permission assignment language—indicates cross-system consistency requirement between AFT and User Management modules for permission workflows.

- **Geographic Contact Configuration**: FTP User Request form (ID 2010) now directs BC and Ontario credit unions to ServiceNow link (versus fax), while Saskatchewan credit unions have fax removal—suggests regional routing logic or integration with ServiceNow ticketing system for certain client segments.

- **Document Library as Master Reference**: The Document Library tracks form revisions and versioning (e.g., form IDs 6257, 2010)—this appears to be the source of truth for user-facing forms and their metadata.

- **Central 1 Support Escalation**: Support contact (1-888-889-7878 Option 1, support@central1.com) is the primary escalation path for Payment Solutions questions—relevant for support integration planning.
