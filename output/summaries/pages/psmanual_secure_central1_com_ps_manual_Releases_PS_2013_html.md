# Payment Solutions 20:13 Release Summary (April 30, 2020)

- **Overview**: Documentation update clarifying Corporate Capture user setup procedures across two management platforms (CPS Admin and User Management), with improved workflow-to-role field correlation.

- **Key Concept - Dual Management Paths**: Corporate Capture users can be provisioned via either CPS Admin or User Management application; documentation now explicitly routes users to the appropriate reference based on their management platform choice.

- **CPS Admin Integration Point**: Users managing Corporate Capture via CPS Admin must reference Form 5936 (Corporate Capture, User Role Combinations) for role assignments and CPS Admin Security Officer configuration with Corporate Capture access permissions.

- **User Management Integration Point**: The User Management application now includes enhanced procedures (Section 9.6, Corporate Capture) that correlate user role fields directly to Corporate Capture workflow options, reducing configuration ambiguity.

- **Critical Data Reference - Form 5936**: Document 5936 serves as the authoritative source for Corporate Capture user role combinations and is required reading for both CPS Admin and User Management implementations; clarity improvements were applied throughout this document.

- **Elimination of Information Redundancy**: Duplicate user management information was removed from Clearing and Deposits Volume (Section 9.6), consolidating guidance and reducing maintenance burden.

- **Support/Escalation**: Client Support Services (1-888-889-7878, Option 1, or support@central1.com) is the documented contact for implementation questions.
