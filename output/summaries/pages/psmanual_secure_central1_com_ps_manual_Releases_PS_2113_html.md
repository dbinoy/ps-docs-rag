# Release 21:13 Summary – May 3, 2021

- **Overview:** Release notes documenting FX Drafts product updates, system requirement changes, and documentation revisions for Western Union's foreign exchange drafts platform.

- **Browser Support Discontinuation:** Internet Explorer 11 support is being deprecated in alignment with Microsoft's end-of-support announcement; future enhancements will no longer target IE11 compatibility.

- **System Requirements Update:** Section 1.2 of FX Drafts documentation was revised with updated system requirements—architects should reference the updated guide for baseline infrastructure/browser specifications when designing integrations.

- **Enrollment Process Clarification:** Client Support (not general contacts) is designated as the single point of contact for new FX Drafts enrollments, establishing a critical dependency for customer onboarding workflows.

- **Documentation Artifacts:** Two user guides (9182 for CU Admin Users, 9183 for Order Creator Users) were published with revisions aligned to system requirement changes—indicates role-based access patterns in the platform.

- **Contact Information Refresh:** Contact details for Western Union were updated in Section 1.3; architects should verify current endpoints before integrating with Western Union backend services.

- **Support Channel:** Client Support Services (1-888-889-7878 Option 1, support@central1.com) is the integration touchpoint for technical issues and enrollment inquiries.
