# Payment Solutions 22:12 Release Summary (April 7, 2022)

- **Overview**: Release notes documenting documentation updates to Currency Ordering procedures and Cash Services reporting capabilities within the Payment Solutions manual.

- **New Reporting Capability**: Two new reports added to Cash Services reporting module—Order History Report and User Audit Report—requiring integration with Currency Ordering workflows; procedures documented in new Chapter 8.

- **Currency Ordering Process Updates**: Cross-referenced reporting capabilities in Section 1.1 (Overview); removed contaminated/mutilated coin reference from Resources/Services contact matrix (Section 1.2), indicating potential relocation or deprecation of that service pathway.

- **Regulatory/Compliance Requirement**: Revised procedures for returning contaminated and/or mutilated Canadian coin (Section 11.10) suggest changes to coin handling validation rules or return workflow logic that architects should validate against Canadian central banking requirements.

- **Authentication Enhancement**: Password requirement details added to Section 6.5 (Resetting User's Password), indicating tightened identity/access control policies that may impact user provisioning and credential management systems.

- **Documentation-Only Release**: No code changes or system architecture modifications implied; this is a documentation baseline update—architects should request corresponding technical specifications if system functionality actually changed.
