# Release Summary: PS 23:09 (May 23, 2023)

- **Purpose**: Documentation update to Form 2725 (Digital & Payment Services Class A Fee Schedule) to publish previously undocumented Log File fee across multiple service modules.

- **Scope & Constraints**: Update applies exclusively to BC and Ontario Class A credit unions; no fee changes or new fees introduced—this is a documentation/transparency update only.

- **Affected Fee Schedule Sections**: Log File fee now documented in four operational fee categories:
  - Section 6.2: Forge Retail Optional Features
  - Section 6.4: Forge Small Business Optional Services
  - Section 8.1: Operating Fees (MDO - Merchant Data Only)
  - Section 9.2: Operating Fees (MDI - Merchant Data Integration)

- **Document Status**: Form 2725 marked as "Revised" in Document Library; supersedes previous version without the Log File fee entries.

- **Integration Consideration**: Architects should ensure fee calculation engines and rate card systems reference the updated Form 2725 sections; Log File fee may impact billing data flows for Forge Retail, Forge Small Business, MDO, and MDI modules.

- **Support & Change Control**: Questions routed to Client Support Services (1-888-889-7878, Option 1) or support@central1.com; implies potential customer inquiry spike requiring communication planning.
