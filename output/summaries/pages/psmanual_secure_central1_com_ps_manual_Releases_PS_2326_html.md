# Payment Solutions Documentation Portal - Release 23:26 Summary

- **Purpose**: Announcement of migration from secure site to Client Centre platform for Payment Solutions documentation, effective November 10, 2023

- **Documentation Migration**: All Payment Solutions Manual content, release notes, client support materials, and product information transitioning from legacy secure site (`psmanual.secure.central1.com`) to centralized Client Centre portal; access controls and manual permissions remain unchanged during migration

- **Notification System Integration**: New Client Centre email notification system (active November 3, 2023) enables opt-in subscriptions for fraud alerts, regulatory updates, manual/form releases, and Central 1 news; automatic enrollment for users with pre-November 1 Client Centre access, with preference management available

- **Content Repository Structure**: Policies and Procedures documentation relocated from secure site to Client Centre's Forms and Manuals page; Payment Solutions Manual also accessible via separate Payments Centre platform under Resources section

- **Multi-Portal Architecture**: Payment Solutions content spans three access points—Client Centre (primary), Payments Centre (alternate with supplementary user guides/overviews), and legacy secure site (deprecated)—requiring clear routing logic and cross-portal consistency

- **Data Persistence Constraint**: User notification preferences and access permissions must be migrated without disrupting existing manual access controls; no changes to role-based access models

- **Support & FAQs Integration**: Client Support Services (1-888-889-7878, support@central1.com) and Client Centre FAQ section serve as escalation paths for migration-related issues; Relationship Manager contact required for account-level concerns
