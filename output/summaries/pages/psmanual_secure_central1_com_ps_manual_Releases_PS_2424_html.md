# Summary: Digital & Payment Services Class A Fee Schedule Update (24:24 – May 29, 2024)

- **Document Update**: Form 2725 (Digital & Payment Services Class A Fee Schedule) was revised to reflect changes in ATM/POS acquiring fees and network connectivity charges for BC and Ontario Class A credit unions only.

- **Scope Constraint**: Form 2725 applies exclusively to Class A credit unions in BC and Ontario; equivalent fee schedules for other institution types or regions are separate documents.

- **Fee Structure Changes**:
  - Dual Acquiring PLUS/VISA CU ATMs: Added PLUS Acquiring fee component; adjusted basis points for International Acquirer Service fee; introduced Visa Acquirer Data Processing fees for international transactions
  - Cirrus Network: Corrected ATM Support fee calculation for International Acquired Transactions; added Mastercard Network Connectivity fee

- **Pass-Through Fee Model**: All updated fees are pass-through charges already in effect as of the release date—no implementation delay; fees flow directly from network operators (Visa, Mastercard, PLUS, Cirrus) without markup changes.

- **Integration Point**: Form 2725 feeds into downstream fee calculation engines and billing systems; any system processing ATM/POS transactions for these credit unions must reference the updated fee table in Section 2.3 for accurate settlement and member billing.

- **Contact/Governance**: Updates managed by Client Support Services (1-888-889-7878 Option 1); questions about fee interpretation or implementation should route through official support channels.
