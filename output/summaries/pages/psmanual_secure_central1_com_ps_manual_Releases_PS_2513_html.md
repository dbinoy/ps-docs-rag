# Payment Solutions Release Summary: 25:13 (May 29, 2025)

- **Release scope**: Updates to Payment Services Class A Fee Schedule (Form 2725) applicable to BC and Ontario Class A credit unions only—not a broader system release.

- **Digital banking fee management transition**: Digital banking fees removed from Form 2725 following operational handoff to Intellect Design; no fee changes occurred during transition, but this represents a change in responsibility boundaries and support ownership (Intellect Client Partner or 1-888-889-7878 Option 2).

- **FX Drafts platform fee restructuring**: Convera updated monthly core platform fee amount and structure for drafts processing across Canadian Dollars, US Dollars, and Foreign Currencies (Section 1.7)—effective June 1, 2025; integration point with Convera requiring coordination on fee synchronization.

- **Interac e-Transfer real-time transaction removal**: Central 1 eliminated "Originating Me-To-Me" and "Originating My-Business-To-My-Business" transaction types (Section 2.14) as these real-time pathways are no longer facilitated; clients must handle workflow adjustments for these payment flows.

- **Token fees consolidation**: Chapter 9 restructured to isolate token-related fees only, suggesting possible separation from broader fee categories for cleaner data model organization.

- **Support continuity**: Dual support channels maintained (Client Support: Option 1 via 1-888-889-7878 or support@central1.com) for fee schedule questions, indicating no architectural changes to support ticketing systems.
