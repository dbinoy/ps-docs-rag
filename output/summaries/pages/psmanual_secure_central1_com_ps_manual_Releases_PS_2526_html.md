# PS 25:26 Release Summary (November 20, 2025)

- **Release scope**: Introduction of a new Data Hub documentation Part covering analytics platform access and usage capabilities for Payment Solutions users.

- **Data Hub platform capabilities**: Interactive analytics engine that visualizes financial data with multi-level filtering, chart/table rendering, peer comparison benchmarking (anonymized institutions of similar size/product usage), and multi-format export functionality.

- **Key user interface**: Payments Dashboard within Data Hub—the primary analytics interface for authorized users to perform analysis and generate insights.

- **Access control model**: Role-based authorization required; documentation indicates "authorized users" can access Data Hub features, suggesting permission/entitlement validation is a prerequisite.

- **Data comparison mechanism**: Enables performance benchmarking against anonymized peer cohorts segmented by institution size and product usage patterns—requires aggregated, anonymized reference datasets.

- **Export capabilities**: Multi-format output support (specific formats not detailed in this release note) for exporting filtered analytics results.

- **Documentation artifact**: New Part added to ps_manual documentation hierarchy under Releases section—indicates this is significant enough for dedicated technical documentation rather than release notes alone.
