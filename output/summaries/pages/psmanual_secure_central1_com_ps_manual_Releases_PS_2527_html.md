# Payment Solutions Class A Fee Schedule Update (Form 2725) - November 21, 2025

- **Scope**: Updates to Payment Services Class A Fee Schedule (Form 2725) applicable exclusively to BC and Ontario Class A credit unions; represents fee structure changes effective across multiple service categories.

- **FX Notes Vendor Migration**: Transition from legacy FX Notes Plus provider to Globex 2000 as new vendor; architectural constraint that drafts are no longer supported through new vendor, requiring removal of subsection and related fee structures.

- **ATM/POS Network Fee Corrections**: Updated Cirrus Network fees including corrected "Mastercard Network Connectivity" fee; added VISA System Access fee and Mastercard QMR-Acquirer Volume Assessment fee (retroactive to 2024)—indicates fee data structure must support multiple network-specific assessment types.

- **Interac e-Transfer Fee Complexity Migration**: Fees moved out of published fee schedule into monthly billing reports (effective mid-December 2025); data flow shift from static form-based reference to dynamic transaction-level billing reports—integration point with billing system required.

- **FX Currency Delivery Service Discontinuation**: Removed home delivery, cash letters, and cheque collection fees due to Globex 2000 service constraints; architectural implication that fee schedule must be decoupled from vendor service capabilities.

- **Billing Report Dependency**: Monthly billing reports become authoritative source for Interac e-Transfer fees starting mid-December 2025; requires integration between transaction processing, rate calculation engine, and billing/reporting system.

- **Support & Escalation**: Client account executive engagement required for Interac e-Transfer fee inquiries; indicates fee complexity warrants direct account management rather than self-service reference.
