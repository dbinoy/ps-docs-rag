# Payment Solutions Manual Updates – October 2020

- **Purpose**: Release notes documenting procedural and documentation updates to the Payment Solutions Manual for October 2020, primarily affecting credit union operations and form management.

- **FX Notes Enhancement**: New capability to attach notes or small files to individual transaction receipts within the FX Notes Plus module—requires UI/storage considerations for receipt-level metadata and file attachment handling.

- **Form Lifecycle Management**: Form 5330 (PaymentStream AFT User Request) transitioned to obsolete status; user access/permissions management migrated to dedicated User Management system—indicates architectural shift from form-based to system-based access control.

- **Form 6257 Retention**: PaymentStream AFT User Request for Business Members (Form 6257) remains available as editable template for internal credit union use—represents dual-path approach where system manages primary workflows while forms support secondary, manual processes.

- **Data Dependencies**: Contact information updates for Saskatchewan and Atlantic credit unions across Document Library forms—implies contact master data feeds or manual maintenance processes that may impact downstream integrations.

- **CEBA Loan Administration**: Form 9140 updated with clarification on fee payment viewing location (Section 7.3)—points to potential reporting/query interface dependency where accredited credit unions access historical loan fee data.

- **User Management System Integration**: Confirms existence of separate User Management module that now owns access/permission workflows previously handled by form submissions—critical architectural dependency for any identity/authorization enhancements.
